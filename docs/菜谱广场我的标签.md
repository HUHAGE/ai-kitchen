# 菜谱广场"我的"标签功能

## 功能说明

在菜谱广场中，用户自己创建的菜谱会显示一个"我的"标签，方便用户快速识别自己的菜谱。

## 实现细节

### 1. 修改菜谱广场数据源 (store.tsx)

**修改前：**
- 菜谱广场只显示其他用户的菜谱
- `publicRecipesList` 过滤掉了当前用户的菜谱

**修改后：**
- 菜谱广场显示所有菜谱（包括用户自己的）
- `publicRecipesList` 包含所有有效菜谱

```typescript
// 菜谱广场包含所有菜谱（包括我的菜谱）
const publicRecipesList = validRecipes;
```

### 2. 添加"我的"标签 (pages/Recipes.tsx)

在菜谱卡片的右上角，为用户自己的菜谱添加"我的"标签：

```tsx
<div className="absolute top-2 right-2 flex gap-2">
  <Badge>{recipe.difficulty}星</Badge>
  {/* 在菜谱广场上为我的菜谱添加"我的"标签 */}
  {recipeTab === 'public' && user && !user.isGuest && recipe.userId === user.id && (
    <Badge color="primary">我的</Badge>
  )}
</div>
```

### 3. 显示条件

"我的"标签只在以下条件下显示：
- 当前在"菜谱广场"标签页 (`recipeTab === 'public'`)
- 用户已登录且不是访客 (`user && !user.isGuest`)
- 菜谱是当前用户创建的 (`recipe.userId === user.id`)

## 用户体验

- 在"我的菜谱"标签页：只显示用户自己的菜谱，不显示"我的"标签（因为都是自己的）
- 在"菜谱广场"标签页：显示所有菜谱，用户自己的菜谱会有绿色的"我的"标签

## 视觉效果

- "我的"标签使用 `primary` 颜色（绿色系），与应用主题色一致
- 标签位于菜谱卡片右上角，与难度星级标签并排显示
