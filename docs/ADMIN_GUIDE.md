# 管理员页面使用指南

## 访问方式

访问管理员页面：`/#/admin`

## 登录

- 默认密码：`huha2026`
- 输入密码后点击"登录"按钮

## 功能说明

### 菜谱列表

管理员页面会显示所有菜谱的列表，包含以下信息：

- **菜谱名称**：显示菜谱名称和缩略图
- **分类**：菜谱所属分类
- **难度**：用星级表示（1-5星）
- **食材数**：该菜谱包含的食材数量
- **步骤数**：该菜谱的烹饪步骤数量
- **创建者**：创建该菜谱的用户ID（显示前8位）

### 删除菜谱

1. 点击菜谱行右侧的"删除"按钮
2. 确认删除操作（会显示"确认删除"和"取消"按钮）
3. 点击"确认删除"完成删除，或点击"取消"放弃操作

### 退出登录

点击页面右上角的"退出登录"按钮即可退出管理员模式。

## 修改密码

如需修改管理员密码，请编辑 `pages/Admin.tsx` 文件中的 `ADMIN_PASSWORD` 常量：

```typescript
const ADMIN_PASSWORD = 'your_new_password';
```

## 注意事项

- 删除操作不可恢复，请谨慎操作
- 删除菜谱会同时删除相关的食材关联和步骤信息
- 建议定期备份数据库
