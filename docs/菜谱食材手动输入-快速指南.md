# 菜谱食材手动输入 - 快速指南

## 概述

菜谱食材现在完全独立于冰箱管理，所有食材都通过手动输入，不再与冰箱食材关联。

## 使用方法

### 1. 创建新菜谱

1. 进入"菜谱"页面
2. 点击"创建菜谱"按钮
3. 填写基础信息（菜名、分类、难度等）
4. 在"所需食材"部分，点击"添加食材"按钮
5. 输入食材信息：
   - 食材名称：例如"鸡蛋"
   - 数量：例如"3"
   - 单位：例如"个"
6. 继续添加其他食材
7. 填写烹饪步骤
8. 点击"保存菜谱"

### 2. 编辑现有菜谱

1. 在菜谱列表中找到要编辑的菜谱
2. 点击"编辑"按钮
3. 修改食材信息（名称、数量、单位）
4. 点击"保存菜谱"

### 3. 导入菜谱

1. 进入"菜谱"页面
2. 点击"批量导入"按钮
3. 粘贴 Markdown 格式的菜谱内容
4. 点击"导入菜谱"按钮
5. 系统会自动解析食材信息，不会创建冰箱食材

### 4. 查看菜谱

- 菜谱详情页会显示所有食材的名称、数量和单位
- 烹饪页面会显示所需食材列表
- 不会显示冰箱库存状态

## 与冰箱的关系

### 完全独立

- 菜谱食材不再关联冰箱
- 导入菜谱不会自动创建冰箱食材
- 完成烹饪不会扣减冰箱库存（除非之前有关联）

### 推荐功能

- 首页的"食材齐全"推荐：只针对有冰箱关联的旧菜谱
- "拯救临期食材"推荐：只针对有冰箱关联的旧菜谱
- 新创建的菜谱不会出现在这些推荐中

## 数据兼容性

### 旧数据

- 之前创建的菜谱（关联了冰箱食材）仍然可以正常显示
- 这些菜谱会显示冰箱食材的名称
- 完成烹饪时仍会尝试扣减库存

### 新数据

- 新创建或编辑的菜谱，所有食材都是手动输入
- 不会关联冰箱食材
- 完成烹饪时不会扣减库存

## 优势

1. **简化操作**：不需要先在冰箱中添加食材
2. **更灵活**：可以输入任意食材名称和单位
3. **独立管理**：菜谱和冰箱完全独立，互不影响
4. **更快速**：导入菜谱时不需要创建大量冰箱食材

## 注意事项

1. 手动输入的食材名称要准确，避免拼写错误
2. 单位要统一，例如都用"克"或"个"
3. 如果需要管理冰箱库存，请在"冰箱"页面单独管理
4. 菜谱和冰箱现在是两个独立的功能模块
