# 游客访问我的主页修复

## 问题描述
游客访问"我的主页"菜单时，展示的页面与"我的冰箱"不一致。

## 最终解决方案
游客访问时，直接在导航菜单中隐藏"我的主页"、"我的冰箱"和"今日计划"菜单项，只显示"菜谱大全"。

## 修改文件
- `components/Layout.tsx`
- `pages/Dashboard.tsx`

## 修改内容

### Layout.tsx
```typescript
// 1. 为每个菜单项添加 guestVisible 属性
const navItems = [
  { to: '/', icon: Home, label: '首页', guestVisible: false },
  { to: '/fridge', icon: Refrigerator, label: '我的冰箱', guestVisible: false },
  { to: '/recipes', icon: ChefHat, label: '菜谱大全', guestVisible: true },
  { to: '/plan', icon: Calendar, label: '今日计划', guestVisible: false },
  { to: '/profile', icon: UserCircle, label: '我的主页', guestVisible: false },
];

// 2. 根据用户状态过滤菜单项
const visibleNavItems = navItems.filter(item => {
  if (!user || user.isGuest) {
    return item.guestVisible;
  }
  return true;
});

// 3. 使用 visibleNavItems 渲染菜单
{visibleNavItems.map((item) => (
  // ... 菜单项渲染
))}
```

### Dashboard.tsx
移除了游客检测和 `GuestPrompt` 组件，因为游客无法通过菜单访问此页面。

- ✅ 登录用户可以看到所有菜单项
- ✅ PC 端和移动端菜单都正确过滤
